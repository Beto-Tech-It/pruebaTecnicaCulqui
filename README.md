# Restful Booker API Automation - Cypress + TypeScript

## Endpoints Covered

- Auth – CreateToken
- Booking – GetBooking
- Booking – UpdateBooking

## Test Coverage

- Happy Path
- Unhappy Path

## Technical Implementation

- Cypress with TypeScript
- Base URL configuration
- Environment variables
- Custom command for token creation
- Fixture-based test data
- Strong typing using interfaces

## Run Tests

Install dependencies:
npm install

Open Cypress:
npx cypress open

Run headless:
npx cypress run
